# 贾维斯智能监控系统

一个JARVIS风格的家庭网络实时监控系统，专为iStoreOS软路由设计。

## ✨ 功能特性

- 🌐 **实时网络流量监控** - 上传/下载速度、总流量统计
- 📱 **在线设备管理** - 设备列表、连接状态、实时速度
- 📊 **网络延迟监控** - Ping延迟、丢包率、抖动
- 🖥️ **路由器状态监控** - CPU、内存、温度、运行时间
- 📈 **历史数据分析** - 24小时流量趋势、CPU使用率趋势
- 🎨 **JARVIS风格界面** - 青蓝色霓虹发光、全息投影效果
- ⚡ **实时更新** - 每秒刷新数据
- 💾 **数据存储** - 保存7天历史数据

## 🖼️ 界面预览

贾维斯风格的深色主题，配合青蓝色霓虹发光效果，打造科技感十足的监控界面。

## 🚀 部署说明

### ⚠️ 重要提示

此应用需要部署在**与iStoreOS软路由相同的局域网内**才能正常工作。云端环境无法直接访问本地网络设备。

### 快速开始

详细的部署步骤请查看 [DEPLOYMENT.md](./DEPLOYMENT.md)

简要步骤：

1. **导出代码到GitHub**（在Manus管理界面操作）
2. **克隆到本地网络设备**（树莓派、NAS等）
3. **配置环境变量**（路由器地址、用户名、密码）
4. **安装依赖并启动**
5. **在浏览器中访问**

### 推荐部署设备

- 🍓 **树莓派** - 低功耗，适合24小时运行
- 💾 **NAS** - 如果您有群晖、威联通等NAS设备
- 🖥️ **小型服务器** - 任何能运行Node.js的设备
- 💻 **旧电脑** - 利用闲置电脑作为监控服务器

## 🔧 技术栈

- **前端**: React 19, TypeScript, Tailwind CSS 4, Recharts
- **后端**: Express 4, tRPC 11, Node.js
- **数据库**: MySQL/TiDB
- **数据源**: iStoreOS软路由 API

## 📋 系统要求

- Node.js 18.0.0+
- pnpm 8.0.0+
- MySQL 8.0+ 或 TiDB
- 与iStoreOS软路由在同一局域网

## 🆘 故障排查

### 无法连接路由器

1. 确认设备与路由器在同一局域网
2. 检查路由器IP地址是否为 192.168.100.1
3. 确认路由器启用了SSH/API访问
4. 检查用户名和密码是否正确

### 数据显示为0

1. 检查应用日志查看错误信息
2. 确认路由器可以正常访问
3. 等待几分钟让数据收集服务启动

### 历史数据不显示

系统需要运行一段时间才能积累历史数据，请耐心等待至少1小时。
